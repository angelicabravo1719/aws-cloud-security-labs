# AWS Cloud Security Labs

Hands-on cloud security labs focused on AWS identity management, storage security, monitoring, logging, and network security controls.

## Labs

### Lab 01 – IAM User and Group Management
Creating IAM users and groups while implementing role-based access control.

### Lab 02 – IAM Policy Management
Applying least privilege principles through custom and managed policies.

### Lab 03 – S3 Bucket Security
Configuring secure storage, access controls, and public access restrictions.

### Lab 04 – CloudTrail Monitoring
Monitoring and auditing AWS account activity through CloudTrail logs.

### Lab 05 – Security Groups
Managing inbound and outbound traffic controls for AWS resources.

### Lab 06 – Cloud Security Dashboard
Reviewing cloud security configurations and monitoring controls implemented throughout the lab series.

## Skills Demonstrated

- AWS IAM
- Identity and Access Management
- Least Privilege
- S3 Security
- Cloud Logging
- Cloud Monitoring
